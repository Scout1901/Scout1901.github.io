Course Modules

Space Academy

This project was created using Adobe Captivate as part of a course entitled Rapid Development Tools for Learning. It was an individual project in which I was the sole contributor.

Learning Objectives

By the end of this module students will be able to...
1. Identify the three main criterion for star formation.
2. Explain the concepts of mass and solar mass.
3. Compare and contrast neutron stars and black holes.
4. Predict the “life cycle” of a star given its mass.

Click the link below to watch a walkthrough the course

Project Reflection

This artifact aligns with LTD program goal #1 (objectives 1.1-1.3) because it demonstrates the ability to create a learning experience that promotes student engagement and interaction while utilizing project management skills. This project was challenging to complete for several reasons, the most impactful of which was the time constraints. Learning to use Adobe Captivate under the pressure of a free trial meant that I had to stay organized and motivated to complete my work. I found that creating the astronomy content from scatch provided additional difficulties when trying to stick to a schedule. To mitigate these difficulties I worked to prioritize the most important parts of the course first and then proceeded to add in additional features like music and animations as I had time. If I were to complete this project for a second time I would take additional time on the front end to create a more thorough design plan and storyboard before beginning development.


The Building Blocks of Life

This course was created using a free trial of Articulate Rise outside of a class or work context to learn more about the program. It was an individual project in which I was the sole contributor.

Learning Objectives

By the end of this module students will be able to...
1. Demonstrate understanding of the history of genetics by correctly answering true/false style questions on an in course knowledge check.
2. Identify the key components that make up the structure of DNA and correctly label them on a diagram.
3. Correctly identify the relationship between protein, DNA, and RNA by answering multiple choice and fill-in-the-blank questions on an in course knowledge check.
4. Apply knowledge of DNA to a real life scenario through completion of case study challenge.

To view a PDF overview of the course click here.

Project Reflection

This project represents an understanding of LTD goal #1 (objectives 1.1-1.3) because it shows the ability to design and develop an interactive that effectively meets learning objectives. It also demonstrates the progression of project management skills. Throughout the process of creating this course, I discovered a need for a better organizational and storyboarding strategy. Because it was my first experience using Articulate Rise, I was excited to get started and jumped into building my course without a storyboard or plan to follow. I created several versions of the content that I ended up deleting, and this cost me a lot of time. This ultimately resulted in my free trial ending without the completion of the project. Though the project was just for fun, it was still a disappointing result after all the work I put in. In the future, I plan to take more time on the front end to develop my ideas before jumping into the fun of course creation. I fully intend to complete building this course at some point and hope to implement better time management and planning strategies based on what I’ve learned.
